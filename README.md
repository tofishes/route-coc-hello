# Route-coc简介

基于express.js用于简化前端页面直出流程的框架。<br/>
coc 意为 约定优于配置（convention over configuration）。

route-coc已经规定好 拦截器-路由-页面渲染 这样的一个流程，使用者只需配置一些与业务相关的代码，即可方便的启动一个项目服务。
